# IronMarch Union Content Pass — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wipe the full piece roster, rebuild 17 pieces from the approved spec, rename faction to `ironmarch_union`, add numbered movement (0–4), faction baselines, starting tactic gating, and full bespoke abilities.

**Architecture:** Engine deltas first (movement int, tag counter, income/authority rules, ability triggers, stealth gate) with TDD in `Core.Tests/EditMode`. Then `IronmarchUnionContentFactory` editor script generates assets and rewrites `ContentDatabase`. Finally sweep tests/UI and stub enemies from the new pool.

**Tech Stack:** Unity 6, C# / NUnit EditMode tests, ScriptableObjects under `Assets/_Project/Data/Resources/DeadManZone/`

**Spec:** `docs/superpowers/specs/2026-07-01-ironmarch-union-content-pass-design.md`

**Branch:** `contentpassv1`

```csharp
public static class BuildBoardTagCounter
{
    public static int Count(BuildBoardSet boards, string tagId)
    {
        if (boards == null || string.IsNullOrEmpty(tagId))
            return 0;

        int total = 0;
        foreach (var piece in boards.AllPieces)
        {
            if (PieceTagQueries.HasAnyTag(piece.Definition, tagId))
                total++;
        }
        return total;
    }
}
```
