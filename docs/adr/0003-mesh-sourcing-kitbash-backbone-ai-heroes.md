# 0003 — Mesh sourcing: kitbashed humanoid backbone, AI-generated showcase pieces

Status: proposed (2026-07-10 greenfield art plan)

The infantry roster is built from one shared low-poly humanoid base family (existing Synty-class packs), kitbashed per archetype, so rigging/animation is done once and retargeted everywhere; the ink shader and shared palette (ADR-0001/0002) carry the directed look. AI 3D generation (Meshy/Tripo-class) is reserved for unique showcase pieces — HQs, walkers, bosses — each manually cleaned and matched to the backbone's poly density and proportions. Rejected: all-AI (per-unit rig/cleanup cost plus proportion drift across 46+ pieces), all-store (faction identity capped by available packs; the prettycombat branch already demonstrated the raw-Synty read), all-custom modeling (months before the look is even evaluable).
