# 0004 — Dread run clock: player-paced escalation replaces the fixed 10-fight gauntlet

Status: accepted (2026-07-12 run-structure grilling)

The run's clock is **Dread**, an escalation currency earned only by *winning* fights, with the amount set by the Fight Option taken (easy < normal < hard; losses and boss fights grant none). Fixed Dread thresholds make the next combat a mandatory **Boss Fight** — no fight options that round — and defeating the third boss wins the run. Boss identity (commander + twist + enemy pool: Neutral Militia, Crimson Legion, Ash Wraiths) is decoupled from boss stage: each boss ships three stage loadouts, and the run draws a hidden seeded permutation, so no fixed meta-build against a known boss order exists while the challenge curve stays authored. Consequences accepted: run length is variable (~9-fight rush to ~18+ grind, plus bosses) and everything difficulty-curved (enemy strength, shop tiering, salvage) must key off Dread instead of FightIndex. Rejected: flat per-win Dread (a renamed fight counter), bosses as a chosen fourth fight option (deflates escalation, permits infinite stalling), fixed boss order (solved meta after a few runs), stat-inflated bosses (invisible difficulty), instantly fatal boss losses (save-scum points).
