# docs/archive — SUPERSEDED design documents

**Nothing in this folder is current. Do not design or implement from it.**

These files describe systems that have since been **renamed, replaced or deleted** — Morale as a
run-level resource (now per-unit; Manpower is run health), Gold (now Supplies), 8x2 reserves
(now 6x2), 6 shop slots (now 5), and more. They were still being picked up as if authoritative,
which is exactly the problem this archive exists to stop.

**The single source of truth is `docs/GDD.md`.**

Kept for history only: these files record *why* decisions were made, and the git history depends
on their paths. Read them as archaeology, never as spec.

Still authoritative and NOT archived:
- `docs/adr/` — architecture decision records
- `docs/art/style-bible/` — art direction
- `docs/GDD.md` — the game design
